## **Pipeline Definition**

1. **OBSERVING**: Feature exists in Phenotype, structure being analyzed.
2. **ABSORBING**: Code being decoupled; interface defined.
3. **HARDENING**: Integration tests passing; Registry spec finalized.
4. **DISTILLED**: Merged into `omni-os` core (Genotype).
