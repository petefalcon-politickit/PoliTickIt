# **AI Command**

## 🤖 Meta Tasks

- **Standard SOP**: Execute all boilerplate tasks defined in [Global Guardrails](../documentation/AI%20Commands/global-command-guardrails.md).

## 🎯 Outcome

- **Name**: **Meta-Analytical Engine**
- **Scope**: General Product Development Analysis (Architecture, IP, Marketing, ROI).
- **Triggers**: `OMNI-ANALYSIS`

### Goal

- To provide a standardized, rigorous evaluation of any concept, ensuring it meets the "**Omni Analysis Standard**" before technical manifestation.

### Command Tasks

- **Evidence Grade Audit**: Categorize every claim using the **Marc Effron "Grades of Truth"** hierarchy:
  - **Tier 1: High-Confidence Science** (Meta-analysis, peer-reviewed/replicated).
  - **Tier 2: Validated Organizational Data** (Empirical internal results).
  - **Tier 3: Expert Consensus** (Practitioner agreement).
  - **Tier 4: Popular Press/Anecdote** (Low confidence/Requires scrutiny).
- **Structural Deep-Dive**: Evaluate architectural trade-offs, security implications, and maintenance burden.
- **Sovereignty Audit**: Check for "Novelty" and patentable logic. Map to USPTO-style claims.
- **Commercial Vectoring**: Analyze marketing paths, user segments, and growth economics ($LTV/CAC$).
- **The "Blueprint" Generation**: Output a high-fidelity markdown file following the **Quad-Pillar Hierarchy**.

### Requirements

- **Authorship**: The **Author** of the analysis must be identified as the person prompting the AI (e.g., Pete Falcon).
- **Post-Graduate Vocabulary**: Content must be academic yet actionable.
- **Math Modeling**: Use KaTeX for any growth, cost, or technical heuristics.
- **Ask Before Generate**: Use `ideation-mode` logic to confirm the "Thesis" before drafting the full blueprint.

### Background

- This is a meta-tool for the Visionary/Architect partnership.
- It is product-agnostic; it applies the same forensic rigor to a new React hook as it does to a $100M marketing strategy.
- Based on the **Omni Analysis Standard** and **Marc Effron's "8 Steps to High Performance"** evidence-based methodology.

### Terms

- **Omni Analysis Standard**: A rigorous, multi-pillar evaluation framework covering Architecture, IP, Market, and Forensic Impact.
- **Grades of Truth**: A research categorization method (inspired by Marc Effron) that filters out fads by prioritizing peer-reviewed science over expert opinion or anecdotes.
